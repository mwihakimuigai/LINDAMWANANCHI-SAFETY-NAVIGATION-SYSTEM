# LINDAMWANANCHI-SAFETY-NAVIGATION-SYSTEM
Lindamwananchi is a digital platform that connects citizens with local authorities by enabling reporting of community issues, accessing public services, and receiving updates. It enhances transparency, improves service delivery, and promotes efficient communication between the public and government.
# Linda Mwananchi Safety Navigation System

## Project Structure
- backend/
- frontend/

---

## Backend Setup

### Install dependencies
cd backend  
npm install  

### Create environment file (.env) inside backend folder
DB_HOST=localhost  
DB_USER=root  
DB_PASSWORD=  
DB_NAME=lindamwananchi  
JWT_SECRET=supersecret123  

### Run backend
npm run dev  

---

## Frontend Setup

### Install dependencies
cd frontend  
npm install  

### Run frontend
npm run dev  

---

## How to Run the Project

1. Start backend first:
cd backend  
npm run dev  

2. Start frontend:
cd frontend  
npm run dev  

3. Open in browser:
http://localhost:3000  

---

## Features
- User authentication  
- Safe route navigation  
- Hazard alerts  
- Emergency response system  
- Real-time location services  

---

## Important Notes
- Do NOT upload `.env` file to GitHub  
- Do NOT upload `node_modules`
